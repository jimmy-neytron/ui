# Доступность

Компоненты используют семантический HTML, стабильные SSR-safe id и видимый `:focus-visible`.

## Рекомендации

- Передавайте `label` или native `aria-label`.
- Для кнопки только с иконкой обязательно задавайте `aria-label`.
- Ошибку передавайте через `error`, пояснение — через `hint`.
- После изменения цветов проверяйте контраст обеих тем.

## Клавиатура UiSelect

| Клавиша | Действие |
|---|---|
| `ArrowDown` / `ArrowUp` | Открыть меню или перейти к опции |
| `Home` / `End` | Первая или последняя доступная опция |
| `Enter` / `Space` | Открыть или выбрать |
| `Escape` | Закрыть и вернуть focus |
| `Tab` | Закрыть без блокировки навигации |

Disabled-опции пропускаются. Select реализует combobox/listbox contract.

::: warning
Итоговое соответствие WCAG 2.2 AA зависит также от контента и цветовых переопределений приложения.
:::
